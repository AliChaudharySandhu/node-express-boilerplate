# Nodejs Microservice API starter project

This is a starter project for developing express api micr-service in Node.js using ES9..

## Getting Started

### Clone the repo and make it yours

```bash
```

#### Install dependencies

```bash
yarn
```

## Running Locally

```bash
yarn dev
```

## Running in Production

```bash
yarn start:prod
```
